Data layer example 1

This example demonstrates the layer option added in Galleria 1.2.5. You can add any HTML as layer data and Galleria will insert it in a layer above the image and follow any transitions in the gallery.

The layer is placed and stretched above the entire image, so you can position any element inside relative to the current image.

If you like to position the layer relative to the container regardless of image size, you can set layerFollow to true.

Please view source to see how we defined the layer in the data model using JSON.
